# V13.1 Interactive Candlestick Chart

The company-profile candlestick chart now supports:

- Quick ranges: 1D, 1W, 1M, MTD, YTD, 1Y, and All
- Mouse/touch dragging to move backward and forward through history
- Wheel/trackpad zoom centered around the pointer
- Hover crosshair with exact date, open, high, low, close, volume, and event classification
- Visible-period return and exact visible date range
- Research event markers retained at every zoom level

No database migration is required for this chart update.
